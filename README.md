JourneySync is a full-stack travel booking and management platform designed to simplify trip planning and travel experiences. The platform allows users to book tickets, manage profiles, access travel assistance, and handle payments through an interactive and user-friendly interface.

Features:

User Authentication & Signup,
Travel Booking System,
Passenger Details Management,
Wallet & Payment Integration,
AI Chatbot Support,
User Profile Management,
Booking History Tracking,and
Responsive Frontend Design.

Tech Stack:

Frontend:
HTML
CSS
JavaScript

Backend:
Node.js
Express.js

Database:
MongoDB / SQL (update according to your database)

Tools & Technologies:
Git & GitHub
REST APIs

Installation & Setup

Clone the repository:
git clone https://github.com/lakshh29/journeysync.git

Navigate into the project folder:
cd journeysync

Install dependencies:
npm install

Run the project:
npm start
